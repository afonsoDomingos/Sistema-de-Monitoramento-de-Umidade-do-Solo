Documentação AquaFlow

O guia completo da plataforma de agricultura inteligente e sustentável para Moçambique.

Stack Tecnológica

Frontend Vanilla

Construído com HTML5 semântico e CSS3 moderno (Custom Properties e Flexbox).

JavaScript (ES6+)

Lógica de simulação, manipulação de DOM e gestão de estado local puramente em JS.

Chart.js

Biblioteca utilizada para renderização dos gráficos de histórico de umidade em tempo real.

Lucide Icons

Conjunto de ícones vetoriais leves e consistentes para interface amigável.

PWA Technology

Service Workers e Manifest para instalação como aplicativo nativo e cache offline.

Integração de APIs Externas

O AquaFlow consome dados reais através de serviços globais de alta precisão:

Open-Meteo API

Utilizada para obter dados meteorológicos precisos de Moçambique sem necessidade de chaves de API complicadas.

  • Endpoint: api.open-meteo.com/v1/forecast
  • Parâmetros: Latitude/Longitude, Current Weather, Daily Forecast.
  • Frequência de Sync: Cada 10 minutos (600,000ms).
Exemplo de Requisição (Fetch) 
fetch("https://api.open-meteo.com/v1/forecast?latitude=-25.96&longitude=32.57¤t=temperature_2m,relative_humidity_2m&daily=weather_code&timezone=auto")
  .then(res => res.json())
  .then(data => console.log(data));

Guia de Integração API

O dashboard está preparado para se conectar a dispositivos IoT. O seu microcontrolador deve enviar um payload JSON no seguinte formato:

sensor_payload.json 
{
  "location_id": "maputo_field_01",
  "moisture": 68.5,
  "pump_status": true,
  "timestamp": "2026-02-06T12:55:00Z"
}
